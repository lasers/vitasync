```bash
Usage: [OPTION]...
  vitasync is an non-interactive bash script using curl, dos2unix,
  pkg2zip, rsync, sha256sum, unzip, and other tools to make it easy
  to keep your folder synchronized with offline content and updates.

Arguments
  -c, --content DIR  (example: ~/Content)
  -u, --updates DIR  (example: ~/Updates)
  -X, --extract DIR  (example: ~/Extracted)
  -f, --filter STR   (example: "nice bird")
  -h, --help         (show this for options)
  --sync             (sync contents & updates)
```
